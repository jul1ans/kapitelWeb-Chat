# kapitelWeb-Chat

## Setup

### clone this repo
```
git clone git@github.com:jul1ans/kapitelWeb-Chat.git
```

### init the project

install the modules
```
npm i
```

build the project
```
grunt build
```

run mongodb in an other terminal
```
mongod
```

start and love it
```
grunt serve
```
